# AI Travel Concierge — Front-end

React front-end for an AI-powered travel concierge built with Claude.

## Stack
- React + Vite
- Claude API via Python back-end
- react-markdown for message rendering
- Real-time cost tracking

## Setup
npm install
npm run dev
